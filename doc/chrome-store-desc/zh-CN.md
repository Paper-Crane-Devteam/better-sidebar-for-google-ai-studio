想找回上周写的那个完美 Prompt 却无从下手？

BETTER SIDEBAR FOR GOOGLE AI STUDIO 专为开发者和提示词工程师打造，旨在大幅提升你的工作效率。我们将原本简单的列表转化为一个功能强大的知识库，让你的 AI 工作流井井有条。

🔥 最新更新 (v1.1.0)
• 📝 Prompt 提示词库：保存并管理你的常用提示词，支持变量模版、文件夹和标签分类。
• 🔍 深度搜索：不仅搜标题，更能全文检索所有历史对话内容。
• 🌐 多浏览器支持：现已支持 Firefox 和 Edge。
• ⚡ 体验优化：侧边栏新增 Dashboard 快捷入口，UI 细节打磨。

🚀 核心功能

📂 文件夹管理（拖拽式）
创建多级文件夹，按项目、客户或主题对对话进行分类。只需拖拽即可整理，保持工作区整洁。

📝 提示词库 (Prompt Library)
构建你的私人 Prompt 库。支持使用变量（如 {{topic}}）创建动态模板，一键插入到对话中。

🔍 深度搜索
告别无尽的滚动。支持对所有历史消息进行全文检索，瞬间找到你需要的任何内容。

🏷️ 智能标签系统
为对话添加自定义标签。无论是“代码调试”、“文案写作”还是“API 测试”，一键过滤，快速定位。

📊 时间轴视图
通过“今天”、“昨天”、“上周”的时间轴视图回顾你的工作轨迹，快速接续之前的思路。

⭐ 收藏与置顶
将最常用的 Prompt 或重要对话置顶，随时调用。

🌗 完美融合
原生级的设计体验，自动同步 AI Studio 的深色/浅色模式。

💾 数据备份与迁移
支持导出和导入数据（SQL 文件），确保你的整理成果安全，或轻松迁移到其他设备。

🔒 隐私优先承诺

• 完全本地化：我们使用浏览器内置的 SQLite 数据库存储你的文件夹结构。
• 无云端同步：你的文件夹、标签和备注“不会”发送到我们的服务器。
• 无数据训练：我们绝不会使用你的对话内容来训练 AI 模型。
