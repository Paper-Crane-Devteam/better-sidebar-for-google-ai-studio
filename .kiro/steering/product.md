# Product Overview

Better Sidebar for Gemini & AI Studio is a browser extension (Chrome, Firefox, Edge) that replaces the native sidebar on Google AI Studio, Google Gemini, and ChatGPT with an enhanced version featuring folder organization, tagging, full-text search, a local prompt library, and conversation export.

All user data is stored locally in an embedded SQLite WASM database inside the browser. No data leaves the device.

Key capabilities:
- Hierarchical folder organization with drag-and-drop (react-arborist)
- Full-text search across conversation messages (FTS5 trigram)
- Local prompt library with variable template support
- Tags, favorites, and timeline views
- Conversation export (Markdown, Text, JSON) and Google Takeout import
- Real-time sync via HTTP request interception (ajax-hook)
- Theme sync with host platform (light/dark)
- Multi-platform: AI Studio, Gemini, ChatGPT (Claude planned)
- i18n: English, Spanish, Japanese, Portuguese, Russian, Chinese (Simplified & Traditional)
